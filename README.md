
2. **Navigate to the Project Directory:**
    ```sh
    cd CAREHEALTH
    ```

3. **Install Dependencies:**
    ```sh
    npm install
    ```

4. **Start the Development Server:**
    ```sh
    npm start
    ```

5. **Open Your Browser:**
    Navigate to 'http://localhost:5173/` to experience CAREHEALTH live!
